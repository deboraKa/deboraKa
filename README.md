## 💻 Sobre Mim

Sou uma profissional da tecnologia com base sólida em **Engenharia de Software** e **Sistemas de Banco de Dados**. Meu foco é transformar grandes volumes de dados em ativos estratégicos através de arquiteturas eficientes, além de possuir competências no desenvolvimento de interfaces funcionais para consumo desses dados.

* **Formação:** Graduada em Banco de Dados e Pós-graduada em Engenharia de Software.
* **Expertise em Dados:** Domínio de todo o ciclo de vida (ETL, Modelagem Relacional, Analytics) e visualização avançada.
* **Desenvolvimento:** Sólidos conhecimentos em Python e SQL, com base em tecnologias Web (JS/HTML/CSS) para suporte a aplicações e integração de sistemas.
